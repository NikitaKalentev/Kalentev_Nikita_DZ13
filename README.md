## Описание проекта

Python-скрипт для взаимодействия с API антивирусной системы VirusTotal. 
Скрипт выполняет авторизацию по API-ключу, отправляет запрос к API и выводит 
результаты сканирования файла в формате JSON.

## Функциональность

- Авторизация в VirusTotal API v3
- Получение отчета о файле по его хэшу (MD5, SHA-1, SHA-256)
- Вывод полного JSON ответа в консоль
- Парсинг и вывод статистики сканирования
- Обработка ошибок и исключений
